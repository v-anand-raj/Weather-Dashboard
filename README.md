# 🌦️ Weather Dashboard

A responsive Weather Dashboard built using **HTML**, **CSS**, and **JavaScript** that displays real-time weather information for any city using the Open-Meteo APIs.

## Features

* Search weather by city name
* Current temperature and weather condition
* 24-hour hourly weather forecast
* 7-day weather forecast
* Feels Like temperature
* Humidity
* Wind speed
* Visibility
* Atmospheric pressure
* Precipitation probability
* Sunrise & Sunset timings
* Air Quality Index (AQI)
* Dynamic weather icons
* Automatic day/night background themes
* Responsive and modern glassmorphism UI
* Smooth animations and hover effects

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Open-Meteo Geocoding API
* Open-Meteo Forecast API
* Open-Meteo Air Quality API
* Font Awesome
